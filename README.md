# Animated Login Form

A modern login & signup UI with glassmorphism design and smooth animations.  
Pure HTML · CSS · JavaScript — no frameworks.

![Screenshot](screenshot.png.png)

## Features

- **Login & Signup** in one card with animated slide transition
- **Glassmorphism card** with backdrop blur and animated background orbs
- **Floating labels** that animate on focus
- **Form validation** with error states and shake animation
- **Password strength indicator** (real-time, 5 levels)
- **Show/hide password** toggle on all password fields
- **Toast notifications** with auto-dismiss and click-to-dismiss
- **Remember Me** saves credentials to localStorage
- **Clickable demo credentials** for quick testing
- **Responsive** — works on mobile, tablet, and desktop
- **Loading state** on submit buttons with spinner
- **Reduced motion** support via `prefers-reduced-motion`

## Tech Stack

HTML5 · CSS3 · Vanilla JavaScript  
Font Awesome 6 · Google Fonts (Inter)  
No Tailwind · No TypeScript · No dependencies

## Quick Start

```bash
npx serve .
# or
php -S localhost:8000
```

Then open `http://localhost:3000` (or `8000`).
